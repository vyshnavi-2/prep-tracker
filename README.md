# TrackPrep – Placement Preparation Tracker

A full-stack web application designed to help students track their placement preparation including DSA, core subjects, and system design.

##  Features
- JWT-based authentication
- Track DSA problems and progress
- Subject-wise preparation tracking
- System design topic management
- Task planner for structured preparation

##  Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

2. Navigate to backend:

   cd backend
